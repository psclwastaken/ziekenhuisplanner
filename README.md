# ziekenhuisplanner
ziekenhuisbezoekplanner

## Runnen

1. Installeer dependencies:

   ```bash
   npm install
   ```

2. Start de server:

   ```bash
   npm start
   ```

3. Open in je browser:

   - http://localhost:3000

Reserveringen worden nu centraal opgeslagen in `data/reservations.json` en gedeeld voor alle gebruikers.
